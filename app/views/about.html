À propos

Cette application (libmeeting) a été développée par Sylvain Machefert en avril 2014, en s'inspirant de l'outil Time is money proposé par Janne Jalkanen.

Cette application a été développée dans le but de s'initier à AngularJS, un framework Javascript permettant d'automatiser les opérations d'interaction au niveau du navigateur. L'application a été générée et développée à l'aide de la suite logicielle Yeoman qui rassemble différents outils faciliant le développement d'applications web (tests automatisés, déploiement simplifié ...)

Le code source du logiciel est disponible sur github.com/symac/libmeeting et est librement réutilisable.
